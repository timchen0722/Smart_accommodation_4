## 二、建議的設計規範（Design Tokens）

沿用現有「日系簡約」色盤，**不換色**；只把它變成唯一來源並補上缺的語意角色。

### 2.1 顏色

| Token | 值 | 用途 |
|---|---|---|
| `primary` | `#4E7FB0` | 主要動作、連結、選中態、房東身分 |
| `secondary` | `#8B7BA8` | 次要強調、平台/後台身分（現 `accent`，保留別名） |
| `success` | `#5B9E73` | 低風險/安全、正向差異、租客身分 |
| `warning` | `#C49A4A` | 中度風險/觀察、需注意 |
| `danger` | `#C4645A` | 高風險、要處理（**跨分頁不得改義**） |
| `ink / ink2 / muted` | `#2A2A2A / #505050 / #9A9490` | 主文 / 次文 / 輔助文 |
| `bg / surface / card` | `#F8F7F5 / #FFFFFF / #FDFCFA` | 頁底 / 面板 / 卡片 |
| `border / border2` | `#E8E4DE / #D4CFC8` | 一般邊框 / 強調邊框 |

每個語意色補一組 **tint / on-tint**（解決問題 #10，統一取現有 badge 的那組）:

| 角色 | tint 底色 | on-tint 文字 |
|---|---|---|
| danger | `#FDECEA` | `#A03028` |
| warning | `#FDF5E4` | `#A07A20` |
| success | `#EAF5EE` | `#3D7A55` |
| info/primary | `#EEF4FB` | `#3D6B96` |
| neutral | `#F2F0EC` | `#505050` |

### 2.3 間距（4px 基準）

`space-1=4 · space-2=8 · space-3=12 · space-4=16 · space-6=24 · space-8=32`

- 頁面上緣留白:**1.6rem**（刪掉 `3_後台分析.py` 的 3.5rem 覆寫）
- 頁面左右留白:沿用 Streamlit `layout="wide"` 預設，不再各頁覆寫

- 卡片內距:`16px 18px`；卡片內元素間距 `space-2`
- 標題到內容:`space-2`；說明文字到內容:`space-3`
